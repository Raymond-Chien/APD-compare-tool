# APD Compare & Edit Tool v2.0 - 專業進階版

這是一套轉為 SSD 測試工程師設計的 **APD (Attribute Profile Document)** 驗證與比對工具。v2.0 版本新增了格式健檢與巢狀結構支援，能更精準地協助準備 OakGate 與 SANBlaze 測試檔案。

## 🌟 v2.0 重點更新 (New in v2.0)

1.  **🔍 APD 格式自動健檢**：
    *   **行尾空白偵測**：自動找出肉眼看不見的隱藏空格（Parser 報錯主因）。
    *   **結構完整性**：檢查標籤是否漏掉 `[END]` 配對。
    *   **佔位符提醒**：偵測是否留有 `ABC12345XYZ` 等未修改的範例內容。
2.  **📦 巢狀層級 (Tree View) 支援**：
    *   **大項目識別**：自動識別 `[ID_CNTLR_DATA]` 等包含子項目的容器。
    *   **視覺縮排**：子項目自動向右縮排並輔以層級連結線 (`└──`)。
    *   **項目統計**：大項目 Badge 會自動顯示包含的子項目數量。
3.  **🏢 專業化 UI 排版**：
    *   **對齊徽章**：所有狀態標籤 (`MATCH`, `DIFF`, `MISSING`) 全部強制靠右排列，介面更整齊。
    *   **深色模式優化**：更清晰的層級配色與高亮效果。
4.  **🛡️ 安全審查工作流**：
    *   **先補齊，後存檔**：將「搬移內容」與「存檔」分開，確保使用者能先 Review 黃色區塊後再導出。
    *   **撤銷保護**：支援一鍵還原該次載入後的所有變更。

## 📖 核心功能與工作流

### 1. 載入與健檢
載入檔案後，程式會自動掃描格式。若有問題會彈出「健康檢查報告」，並在有問題的區塊標示 `⚠️ FORMAT ERR`。

### 2. 結構比對
*   🟢 **MATCH**：兩側標籤內容一致。
*   🟣 **DIFFERENT**：內容有差異，建議手動確認。
*   🔴 **MISSING**：檔案中缺失該標籤，建議補齊。

### 3. 補齊與 Review
*   點擊 **「補齊 A/B」**：將缺失項目從另一側搬移。
*   **Review**：檢查畫面上出現的 **黃色 (MERGED)** 區塊。
*   **還原**：若補齊錯誤，點擊「全部取消」或個別區塊的「❌ 取消」。

### 4. 存檔匯出
確認無誤後，點擊 **「💾 另存 A/B」**。儲存時會保留您的手動編輯內容。

## ⚠️ 工程開發建議
*   **換行符號**：建議使用 **UNIX (LF)** 換行格式以維持最佳相容性。
*   **隱藏空白**：本工具會主動偵測空格，請務必依據 `⚠️ FORMAT ERR` 提示修復之。
*   **層級關係**：若標籤名稱重複，工具仍會依照檔案順序進行區塊化配對。
