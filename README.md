# APD Compare & Edit Tool v2.1 - 專業進階版

這是一套轉為 SSD 測試工程師設計的 **APD (Attribute Profile Document)** 驗證與比對工具。v2.1 版本解決了捲動同步與層級顯示的核心 Bug，提供更強健的操作體驗。

## 🌟 v2.1 重點更新 (New in v2.1)

1.  **⚡ 強健版代碼捲動同步**：
    *   **即時偵測**：重寫同步鎖 (Sync Lock) 邏輯，現在開關同步開關可「即時生效」，不受檔案載入順序影響。
    *   **流暢度優化**：優化事件監聽機制，減少捲動時的延遲感。
2.  **🌲 精確的層級樹狀視圖**：
    *   **深度校正**：修正了 v2.0 中的層級偏移 Bug，確保頂級標籤與子項目（如 `ID_CNTLR_DATA_1`）擁有正確的縮排寬度。
    *   **視覺錨點**：優化了階層連結線 (`└──`) 的對齊方式。

## 🌟 v2.0 回顧功能

1.  **🔍 APD 格式自動健檢**：
    *   **行尾空白偵測**：偵測並標示肉眼看不見的隱藏空格。
    *   **結構完整性**：檢查標籤是否漏掉 `[END]`。
    *   **佔位符提醒**：偵測是否留有 `ABC12345XYZ`。
2.  **🛡️ 安全審查工作流**：
    *   **先補齊，後存檔**：搬移缺失項目後會顯示為 **黃色 (MERGED)**，方便 Review 後再執行「另存新檔」。

---

## 📖 核心功能與工作流

### 1. 載入與健檢
載入檔案後，程式會自動掃描格式。若有問題會彈出「健康檢查報告」，並在有問題的區塊標示 `⚠️ FORMAT ERR`。

### 2. 結構比對
*   🟢 **MATCH**：兩側標籤內容一致。
*   🟣 **DIFF**：內容有差異。
*   🔴 **MISSING**：檔案中缺失該標籤。

### 3. 補齊與 Review
*   點擊 **「補齊 A/B」**：將缺失項目搬移，新項目轉為黃色。
*   **全部取消**：若發現搬移有誤，一鍵撤銷所有變更。

---

## ⚠️ 工程開發建議
*   **換行符號**：建議使用 **UNIX (LF)** 換行格式。
*   **隱藏空白**：請依據 `⚠️ FORMAT ERR` 提示修復空格，避免 OakGate 解析失敗。
