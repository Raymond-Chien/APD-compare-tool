# APD Compare & Edit Tool v1.0 - 使用指南

這是一套專為 SSD 測試工程師開發的 **APD (Attribute Profile Document)** 視覺化比對與編輯工具。它可以幫助您快速對齊 OakGate 或 SANBlaze 的設定檔，偵測缺失標籤，並進行安全的一鍵合併與編輯。

## 🚀 核心功能 (Key Features)

1.  **結構化標籤對齊 (Tag-based Alignment)**：自動依據 `[TAG] ... [END]` 結構進行對齊，即使檔案順序不同也能正確比對。
2.  **智慧內容比對 (Smart Diff)**：
    *   **忽略大小寫** (Case-insensitive) 與行尾空白。
    *   自動過濾空白行，確保內容一致即顯示為 `MATCH`。
3.  **視覺化狀態標示**：
    *   🟢 **MATCH (綠色)**：兩側內容完全一致。
    *   🟣 **DIFFERENT (紫色)**：標題相同但內容有差異。
    *   🔴 **MISSING (紅色)**：該檔案缺失此標籤，並以紅框警示。
    *   🟡 **NEW MERGED (黃色)**：剛從另一側補齊或覆蓋的內容。
4.  **安全補齊與復原機制**：
    *   **一鍵補齊**：快速將 A 檔案缺失的所有項目從 B 檔案搬移過來。
    *   **備份還原**：覆蓋內容時會自動備份舊資料，隨時可按 `❌ 取消補齊` 或 `全部取消` 回滾狀態。
5.  **同步搜尋與捲動**：
    *   **🔍 標籤搜尋**：即時過濾顯示特定標籤 (如 `ID_DATA`)。
    *   **同步捲動**：左右面板同步移動，極速定位差異點。

## 📖 使用教學 (How to Use)

### 1. 啟動工具
直接雙擊開啟目錄下的 `index.html` 即可使用（建議使用 Chrome 或 Edge 瀏覽器）。

### 2. 載入檔案
*   點擊 **「📂 載入 (A)」** 選擇第一個 APD 檔案。
*   點擊 **「📂 載入 (B)」** 選擇要比對的第二個檔案。
*   當兩側檔案都載入完成後，系統會**自動執行比對**。

### 3. 查看差異
*   檢視上方的統計列（不同項目、缺失項目、符合項目）。
*   捲動面板查看內容，差異處會以顏色區塊明顯標出。

### 4. 編輯與同步
*   **手動修改**：您可以直接點擊區塊內容進行文字編輯（自動儲存至記憶體）。
*   **單項複製**：點擊區塊標題旁的 `➡️` 或 `⬅️` 箭頭，將內容同步到另一側。
*   **一鍵補齊**：使用面板頭部的 `補齊右側` 快速填充所有缺失標籤。
*   **取消與還原**：若補齊錯誤，點選黃色區塊的 `❌ 取消補齊` 或頭部的 `全部取消`。

### 5. 儲存結果
*   點擊 **「💾 另存 A」** 或 **「💾 另存 B」** 將編輯後的內容匯出。
*   建議儲存為 `.txt` 格式，並確認換行符為 **UNIX (LF)** 以符合測試機要求。

## ⚠️ 注意事項 (Tips)
*   **隱藏空白**：本工具比對時會自動忽略行尾空白，但儲存時會保留您的編輯格式。
*   **檔案格式**：若 OakGate Parser 報錯，請檢查檔案是否含有非標籤內容，或是缺少必要的 `[END]`。
*   **瀏覽器權限**：若點擊另存新檔沒反應，請點擊網址列旁的「權限」允許下載多個檔案。
